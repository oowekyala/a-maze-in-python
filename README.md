## Maze visualization with Python

This project is a playground to visualize maze generation and
maze solving algorithms. It's my first project in Python.



### Maze generators

Supports the following algorithms:

#### DFS

#### Eller

#### Kruskal

#### Prim

#### Recursive division

#### Sidewinder

#### Wilson



### Maze solvers

#### DFS

#### BFS

#### Dead-end filling

#### Left-/right-hand rule
